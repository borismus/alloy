# Claude Code Project Instructions

## Allowed Tools

This project allows Claude Code to use web search for research and documentation lookups.

allowedTools:
  - WebSearch
  - WebFetch
  - Bash(ls *)
  - Bash(grep *)
  - Bash(find *)
  - Bash(cat *)
  - Bash(head *)
  - Bash(tail *)
  - Bash(wc *)
  - Bash(file *)
  - Bash(pwd)
  - Bash(which *)
  - Bash(echo *)
